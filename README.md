# NFL-analysis
NFL 경기 승패 요인 분석 프로젝트 

## 데이터
- 출처 : nfl_data_py (nflverse)
- 범위 : 2024~2025 정규시즌 544경기

## 분석 결과

### 홈 어드밴티지 
- 홈 승 291 / 원정 승 252 / 무승부 1
- 승률 : 53.5%
- 홈 승리 시 평균 12.3점 차, 패배 시 9.9점 차 
 
→ 홈 팀 승률 50% 이상, 이길 때 더 큰 점수차 // 2시즌 표본 : 추가 검증 필요
